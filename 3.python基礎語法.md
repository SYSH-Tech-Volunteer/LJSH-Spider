# 變數
* 數字不可用於開頭字元。
* 可以使用英文字元、數字或下底線(_)。ex : number_id
* 英文大小寫是有差異的。
* 名稱不可使用python語言保留字詞。ex : int
* 在程式碼上下文中具有唯一性。
```python
x = input("我是變數")
```
